# ArchCoach v0.1.0 — 你的第一次 AI 架构评审

首个公开版本。提交架构方案，6 位 AI 评委（首席架构师 / 技术架构师 / 数据库专家 / SRE / 安全专家 / 面试官）对你的方案进行追问、并行评审、打分与冲突呈现。

## 功能

- 架构方案提交 → AI 追问（6-10 个关键信息缺口）→ 6 角色并行评审 → 结构化评审报告（评分 / 等级 / 风险清单 / 分歧呈现 / 行动清单）
- 自动生成：Mermaid 架构图（服务端语法校验 + 修复回路）、ADR 决策记录合集（MADR 格式）、面试讲解稿（含简历描述）
- **防幻觉机制**：每条评审意见强制引用方案原文，引用无法定位的意见被标记降权
- **确定性仲裁**：任一评委判定"阻塞上线"，总评级最高 C——平均分稀释不掉阻塞项，规则在代码里不在 prompt 里
- 断点续跑：评审中断后继续，已完成的评委不重复计费
- 3 道内置训练题（秒杀 / SaaS 多租户 / RAG 知识库），markdown 文件即题库
- 支持 OpenAI / Anthropic / DeepSeek / OpenRouter / Ollama；**可全本地零 key 运行**（推荐 qwen3:8b+，使用 Ollama 约束解码保证结构化输出）
- 每次评审的 token 用量与估算成本全透明（报告页可见，审计日志全量落库）
- `docker compose up` 一键启动（迁移与题库 seed 自动完成）

## 质量数据

- 80 个单测/集成测试；核心引擎覆盖率 95.7%
- eval 质量门（`pnpm eval`，真实模型）：证据引用率 ≥60%、缺陷方案 blocking 检出率 2/2、好坏方案分数排序正确、生成图 100% 可解析
- 端到端实测（本地 qwen3:8b）：完整评审约 10 次模型调用 / ~43k tokens / 16 分钟；云端模型约 2-4 分钟

## 已知边界

- 单用户本地模式（无认证）——**请勿在开启 LOCAL_MODE 的情况下暴露公网**
- 界面为中文；评审质量取决于所配置的模型（本地 8B 模型可用但意见深度有限）
